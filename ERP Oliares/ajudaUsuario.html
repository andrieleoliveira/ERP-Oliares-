Guia de Uso



Preencha os Campos:

Preencha os campos conforme as indicações, fornecendo as informações necessárias para garantir um registro preciso. Por favor, note que para efetuar o cadastro, é essencial que tanto o nome de usuário (login) quanto a senha ainda não existam no sistema. Certifique-se de criar uma senha robusta, contendo uma combinação de letras, números e símbolos para garantir a segurança da sua conta.

Clique em "Gravar":

Após preencher os campos, clique no botão "Gravar" para adicionar uma nova entrada.

Exclua um Usuario:

Se necessário, utilize o botão "Excluir" para remover um usuario. Confirme a ação quando solicitado.

Volte à Página Inicial:

Retorne à página inicial a qualquer momento clicando no botão "Voltar".

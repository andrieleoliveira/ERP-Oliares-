Guia de Uso



Preencha os Campos:

"Preencha os campos conforme as indicações, inserindo as informações necessárias nos espaços correspondentes para garantir um registro preciso."

Clique em "Gravar" ou "Alterar":

Após preencher os campos, clique no botão "Gravar" para adicionar uma nova entrada. Se precisar fazer alterações, utilize o botão "Alterar".

Exclua uma Entrada:

Se necessário, utilize o botão "Excluir" para remover uma entrada. Confirme a ação quando solicitado.

Volte à Página Inicial:

Retorne à página inicial a qualquer momento clicando no botão "Voltar".
